# نظام الشؤون القانونية - جامعة المنصورة الجديدة

## وصف المشروع
هذا المستودع يحتوي على **النسخة الأولية v0.1.0-alpha** من واجهة نظام الشؤون القانونية لجامعة المنصورة الجديدة.

> ملاحظة مهمة: هذه النسخة **واجهة فقط Front-End Prototype**، وليست نظامًا تشغيليًا كاملًا بعد.

## الحالة الحالية
- نسخة أولية / Prototype.
- واجهات HTML ثابتة.
- لا يوجد Back-End حتى الآن.
- لا توجد قاعدة بيانات حتى الآن.
- لا يوجد نظام تسجيل دخول فعلي أو صلاحيات حقيقية.
- البيانات المعروضة داخل الصفحات بيانات تجريبية/تصميمية لغرض العرض فقط.

## الصفحات المتاحة
- `index.html` — لوحة التحكم الرئيسية.
- `cases.html` — ملفات القضايا.
- `investigations.html` — التحقيقات والشكاوى الداخلية.
- `contracts.html` — العقود والمستندات.
- `agenda.html` — الأجندة والمواعيد.
- `settings.html` — الإعدادات.

## التقنيات المستخدمة
- HTML5
- CSS3
- JavaScript بسيط داخل الصفحات
- Font Awesome Icons
- Google Fonts - Cairo

## طريقة التشغيل محليًا
افتح ملف `index.html` مباشرة من المتصفح، أو شغّل المشروع باستخدام Live Server في VS Code.

## المطلوب في الإصدارات القادمة
- ربط الواجهة بقاعدة بيانات.
- إضافة Back-End API.
- إضافة نظام تسجيل دخول وصلاحيات.
- حفظ وإضافة وتعديل وحذف البيانات فعليًا.
- رفع الملفات والمرفقات.
- البحث والتقارير والطباعة.
